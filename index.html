Chat UX – Rol × nieuw bericht

Matrix met alle toegestane combinaties voor Admin A en Admin B. Het rode bolletje verschijnt alleen bij volgen. Als een admin het bericht leest, verdwijnt het bolletje voor die admin. Mute ziet nooit een notificatie.

Legenda migratie: het rode bolletje is de vervanging van de zwarte kleur uit de oude implementatie. De pin is de vervanging van de blauwe kleur.

Rol × nieuw bericht (matrix)

Scenario Admin A Admin B
A volgt – B volgt
A volgt – B volgt (nieuw bericht)
A volgt – B volgt (na lezen door A)
A volgt – B volgt (na lezen door B)
A volgt – B mute
A volgt – B mute (nieuw bericht)
A mute – B volgt
A mute – B volgt (B wil mute, niet toegestaan)
A mute – B volgt (nieuw bericht)

Bij scenario's met slechts een admin op volgen (A volgt - B mute of A mute - B volgt) is "na lezen" visueel gelijk aan de basisrij zonder nieuw bericht. Een tweede mute is niet toegestaan: er moet altijd minimaal een admin op volgen blijven.

Flow tabel (uit matrix)

Onderstaande flow vertaalt de matrix naar concrete situaties in de praktijk. Zodra Admin A of Admin B het bericht leest, verdwijnt het rode bolletje voor die admin.

Stap Omschrijving Admin A Admin B
1 Bij een nieuw binnenkomend dossier of dossier uit archief staan beide admins op volgen.
2 Er komt een nieuw bericht binnen terwijl beide admins op volgen staan.
3 Admin A leest het bericht; het rode bolletje verdwijnt bij Admin A.
4 Admin B leest daarna ook; het rode bolletje is nu bij beide weg.
5 Admin B zet deze chat op mute; alleen Admin A blijft volgen.
6 Er komt daarna een nieuw bericht binnen; alleen de admin die volgt krijgt een bolletje.
7 Admin A leest het bericht; het bolletje verdwijnt weer.
8 Admin A zet de chat op mute en Admin B blijft volgen.
9 Admin B probeert nu ook op mute te zetten; dit is niet toegestaan omdat altijd minstens een admin moet volgen.
10 Bij een nieuw bericht in deze situatie ziet alleen Admin B het bolletje.
11 Admin B leest het bericht; het bolletje verdwijnt bij Admin B.

3️⃣ Pin tabel

Een pin is altijd persoonlijk per admin en staat volledig los van notificaties en chats.

Status Weergave Betekenis
Actief Reminder: er staat nog een actie open op de chat of het dossier.
Pin uit Geen reminder op de chat of het dossier.

4️⃣ Auto pin (per admin)

Auto pin is een persoonlijke instelling per admin. Als auto pin aan staat, wordt het dossier bij een nieuw bericht automatisch gepind voor die admin. Dit werkt per admin afzonderlijk.

Stap Situatie Admin A Admin B
1 Nieuw bericht, auto pin A = aan, auto pin B = uit
2 Nieuw bericht, auto pin A = aan, auto pin B = aan
3 Nieuw bericht, auto pin A = uit, auto pin B = uit
